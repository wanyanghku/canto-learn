# 粤语卡片 · canto-learn

杨万的粤语（香港话）随身练习卡片。手机浏览器打开即用：点字听音、跟读、翻牌、记进度。

- **九声调表**（第一课重点）：点格听单字、点「整行」听 1→9 调一条龙。
- 22 个卡组：声调 / 入声 / 茶餐厅 / 数字 / 问路 / 常用句…
- 注音 = 粤拼 Jyutping；进度存浏览器本地（localStorage）。
- 音频为机器合成（微软神经语音 edge-tts，zh-HK），个人学习用；抠单字最权威发音配合 [words.hk](https://words.hk/)。

> 内容源自私有笔记 PhD_OS/粤语（生成产物），本仓库只含网页 + 音频。
